# ETL with Python

A short introductory class on ETL (Extract, Transform, Load) using Python.

## Contents

- **Python trabalhado em aula/** — notebooks worked on during class, covering ETL pipelines on Movies and SuperHeroes datasets
- **Exercicio_treino/** — practice exercise using the World Happiness Report 2024 dataset
- **Apresentacao_ETL_Python.pdf** — class slides
- **Guia_Completo_Iniciantes_ETL.docx** — beginner's guide to ETL

## Topics Covered

- Extracting data from CSV files
- Transforming and cleaning data with pandas
- Loading/exporting results to new CSV files
